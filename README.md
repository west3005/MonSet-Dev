## О проекте


MonSet — промышленное IoT-устройство для мониторинга технологических параметров на объектах.

**Аппаратная платформа:**
- MCU: STM32F407VET6 на плате STM32F4VE со встроеной SD-картой (168 MHz, 512KB Flash, 192KB RAM)
- Сеть: W5500 (Ethernet) + SIM7020E (Cat1-A7670X-V1.02) + ESP8266
- Датчики: DS3231 (RTC по I2C), Modbus RTU датчики (415- погружной, ULM31 радарный)
- Хранение: встроенная MicroSD карта в плату STM32F4VE (FAT32)
- Питание: Аккумулятор LiFePo4 24V15A 24V-питание датчиков
            DC-DC 24V-5V - питание ESP8266, W5500, DS3132 и GSM модема
            DC-DC 5V-3,3V - питание STM32F4ve
  
UART1 - Отладочный CH340
UART2 - GSM
UART3 - 415 погружной - RS-485
UART4 - ULM31 радарный - RS-485
UART5 - Блок спутникового модема IRIDIUM RS-232
UART6 - ESP8266 

**Что делает устройство:**
1. Считывает данные с датчиков по Modbus RTU (UART)
2. Синхронизирует время через NTP или DS3231
3. Логирует данные на SD-карту
4. Отправляет телеметрию на ThingsBoard через GSM и Ethernet W5500 (HTTPS и MQTT выбор способа отправки через конфиг)
5. Поддерживает удалённое конфигурирование через WiFi ESP8266 и по статическому IP на СИМ карте по GSM
6. Конфиг хранится на SD карте
7. Использует безопасное удаленное управление с логоном и паролем
8. Использует дискретность опроса датчиков
9. Все опросы датчиков сохраняются на SD карту в файл Bakcup.json
10. Логирование с ротацией
11. Использует отдельную дискрентость отправки Bakcup.json на указаный в конфиге сервер и указанным способом (HTTPS и MQTT)
12. Использует WatchDogs для функций программы и автовосстановления 
13. Веб-интерфейс ESP8266, GSM и Ethernet (W5500) с Runtime - логами и конфигом
14. Скачивание логов по веб интерфейсу
15. При неудачной отправки данные сохраняются в Bakcup.json и отправляются после восстановления связи.
16. После удачной отправки, данные стираются.
17. 


**Конечный автомат App:**
```
INIT → ETHERNET_INIT → NTP_SYNC → MODEM_INIT → SENSORS_POLL → SEND_DATA → SLEEP
                                                      ↑                        |
                                                      └────────────────────────┘
```
